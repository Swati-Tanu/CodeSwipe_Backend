## Run Locally

### Clone this Project

```
https://github.com/Swati-Tanu/CodeSwipe
```

### Install npm Packages

```javascript
npm i --global
```

### Run Server

```javascript
npm run dev
```

### Runs the project in the development mode

[http://localhost:5173](http://localhost:5173)

### Builds the project for production

```javascript
npm run build
```
